# Product Catalog App

A mini product catalog application built with **Next.js 14/15 (App Router)**, **TypeScript**, and **Tailwind CSS**.  
Users can view products, search and filter them, see product details, and add or remove products from their favorites list.

---

## Installation

Clone the repository:

```bash
git clone `‍https://github.com/Parsa81shokati/product-catalog.git‍`
cd product-catalog
```
